# Mi Restaurante — Demo (archivos estáticos)

Instrucciones:
1. Guarda los archivos `index.html`, `styles.css` y `app.js` en la misma carpeta.
2. Abre `index.html` en tu navegador (doble clic).
3. Usa el buscador, filtra por categoría, añade platos al carrito y "realiza pedido".
4. El panel Admin (botón "Mostrar/ocultar Admin") permite añadir nuevos platos. Los datos se guardan en localStorage del navegador.

Notas:
- Es una demo sin backend; los pedidos se almacenan localmente en tu navegador.
- Para producción necesitarías un backend para recibir pedidos, procesar pagos y persistir datos de manera segura.
